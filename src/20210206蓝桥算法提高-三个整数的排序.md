试题 算法提高 三个整数的排序
     
## 资源限制
时间限制：1.0s   内存限制：256.0MB
## 问题描述
　　﻿输入三个数，比较其大小，并从大到小输出。
## 输入格式
　　一行三个整数。
## 输出格式
　　一行三个整数，从大到小排序。
## 样例输入
33 88 77
## 样例输出
88 77 33

## 分析

使用动态数组，写入后使用sort函数排序，然后在使用reverse函数翻转输出即可。

## 代码

~~~cpp

#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;
int main() {
    vector<int> arr;
    int a,b,c;
    cin >> a >>b>> c;
    arr.push_back(a);
    arr.push_back(b);
    arr.push_back(c);
    sort(arr.begin(),arr.end());
    reverse(arr.begin(),arr.end());
    for (vector<int>::iterator it = arr.begin();it!=arr.end();it++) {
        cout << *it << " ";
    }

    return 0;
}

~~~
